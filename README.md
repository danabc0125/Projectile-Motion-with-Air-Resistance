## Project Overview

These projects implement projectile motion simulations using the Euler method and Python's Turtle graphics module, based on user-defined parameters.

- **Project 1** focuses on detailed analysis by visualizing the trajectory, velocity–time (v–t), and acceleration–time (a–t) graphs, along with numerical results.
- **Project 2** compares two projectile simulations under different conditions: with air resistance and without (ideal motion), displaying both trajectories on the same graph.
